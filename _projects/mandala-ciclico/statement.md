_Tobias Gorschin_, 2025.

**Mandala Ciclico**

Este proyecto de arte generativo en 3D explora un ciclo estricto y continuo de vida y muerte. 
El funcionamiento es simple: el código actúa como un reloj que fuerza una regeneración aleatoria de la 
figura cada 8 segundos.

**Desarrollo técnico, estético y conceptual**

El patrón se destruye rápidamente (Decaimiento) y luego se reconstruye lentamente (Regeneración), creando 
un mandala que nunca se detiene. Este énfasis en el proceso y la ocurrencia permanente conecta 
con las reflexiones sobre el arte contemporáneo, como las de César Aira, donde la obra se valora 
no por su objeto final, sino como una fábrica de eventos y un sistema de reglas activo. 
La obra utiliza la rigidez de la programación para explorar la fluidez, manteniendo un 
carácter perpetuo y dinámico.

El fundamento teórico de esta obra es la tesis de Inke Arns sobre el código 
como acto de habla performativo. El software no solo describe la imagen, sino que ejecuta 
la creación en tiempo real: comandos específicos, como startNewCycle(), se vuelven la 
palabra creadora que da vida a una nueva configuración. El código es el motor que gestiona 
la existencia del universo visual. La "performatividad" permite ir más allá, centrándote en el 
código como acto de habla.


**Bibliografía**

Arns, Inke. (2005). El código como acto de habla performativo. Artnodes, UOC.
